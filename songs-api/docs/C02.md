# DO2526-C02: Migración a MongoDB

Este hito consistió en migrar la persistencia de datos de NeDB a MongoDB y dockerizar la solución conjunta.

## Requisitos Completados

- [x] **9. Migración a MongoDB**: Código modificado para usar `MongoDB` en lugar de NeDB.
- [x] **10. Docker Compose**: Configurado `docker-compose.yml` para orquestar la API junto con un contenedor de MongoDB.
  - Persistencia de datos configurada.
  - Healthchecks para la base de datos.
- [x] **CI/CD Extra**: Implementado workflow en GitHub Actions para construir y subir la imagen automáticamente.

## Imagen Entregada
- Repositorio: `danvelcam621/songs-api:DO2526-C02` (o `latest` generado por CI/CD).

## Ejecución y Verificación

### Opción A: Construcción Local
```bash
docker-compose up -d
```

### Opción B: Verificación CI/CD (Desde Imagen Docker Hub)
Para verificar la imagen generada automáticamente por GitHub Actions:

1. Modificar `docker-compose.yml` cambiando la línea `build: .` por `image: danvelcam621/songs-api:latest`.
2. Ejecutar:
   ```bash
   docker-compose up -d
   ```
3. Verificar que la API responde (incluyendo la canción "King Nothing" añadida en el último commit):
   ```bash
   curl http://localhost:8080/api/v1/songs
   ```
